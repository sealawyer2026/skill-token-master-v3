## 简介

Token经济大师是一款专为AI应用设计的Token使用量优化工具。它能自动检测并优化智能体、技能和工作流的Token消耗，在保证功能完整性的前提下，实现最大化的成本节约。

## 核心能力

| 优化维度 | 节省比例 | 说明 |
|---------|---------|------|
| 提示词优化 | 50-80% | 神经网络语义压缩、800+规则 |
| 代码优化 | 75-85% | 结构重组、函数内联、AST优化 |
| 工作流优化 | 40-55% | JSON/YAML压缩、配置精简、路径优化 |

## 四大核心策略

1. **神经网络语义压缩** - 基于语义理解的深度学习压缩
2. **代码结构重组优化** - 函数内联、循环展开、变量合并
3. **自适应学习系统** - 根据输入类型自动选择最优策略
4. **多轮迭代压缩** - 持续优化直至收敛

## 使用方法

```python
# 分析Token使用
python3 -m token_economy_master_v3 analyze /path/to/target

# 执行优化
python3 -m token_economy_master_v3 optimize /path/to/target --output ./optimized

# v3.5.0 超级压缩引擎
python3 v35_engine.py "要压缩的文本" --type prompt --target 0.80

# 智能模式 (自动选择策略)
python3 v35_engine.py "文本或代码" --smart
```
